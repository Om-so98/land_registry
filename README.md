# 🏡 Land Registry Smart Contract (Move)

A blockchain-based land registry system built using the Move programming language.

This project demonstrates how land ownership can be recorded, transferred, and verified securely on-chain using smart contracts.

---

## 🚀 Overview

Traditional land registry systems face challenges such as:
- Fraud
- Double allocation of land
- Manual record tampering
- Slow ownership verification

This smart contract solves these issues by:
- Storing land records immutably on-chain
- Tracking ownership history
- Enabling secure ownership transfers
- Preventing unauthorized modifications
